**华泰保险学习平台 - APP** 

**Scaffold**
----------

    React  + React-Router + Webpack + Amaze/Antd + Cordova

**Running locally in development mode**
----------

    cd react
    npm install
    npm start
    
**Building and deploying in production**
----------

    npm run build

**Run in device**
----------

    cordova build
    cordova run android/ios/browser
